# Notification Operations Center Operations Guide

This guide describes how to manage and audit platform communications from the **Admin Notification Control Panel** in **RishtaJodo Matrimony**.

---

## 1. Operations Center Overview
The Admin Operations Center provides a centralized control panel for all communication channels.
*   **Real-time Analytics**: Tracks outbound alerts counts, success ratios, queue latencies, and daily session costs.
*   **Live Queue Monitor**: Exposes active, pending, and scheduled worker queues.
*   **DLQ Center**: Exposes dead-lettered job logs for manual retry or deletion.
*   **Templates Manager**: CRUD editor for layouts and messages.
*   **Campaign Broadcaster**: Segmented target wizard for marketing broadcasts.

---

## 2. Navigation Tabs

1.  **Control Panel**: Summary metrics cards, cost graphs, and emergency switches.
2.  **Message Templates**: Template creators, subject lines, body inputs, and language localizations.
3.  **Broadcast Wizard**: Select segment targets (gender, KYC verified) and launch multi-channel campaigns.
4.  **Queue Monitor**: Real-time status list of jobs, plus failure logs and retry buttons.
5.  **Delivery Log**: Searchable logs audit table with CSV export actions.
6.  **Provider Health**: Live gateway pings and latency reports.
