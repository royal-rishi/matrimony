# Go-Live Checklist & Verification Manual

Steps for live validation before general launch.

## 1. Domain Configuration
- [ ] Connect Google Workspace and confirm email SPF, DKIM, and DMARC settings are active.
- [ ] Setup SMS DLT sender ID headers for Indian regulations.

## 2. Platform Monitoring & Analytics
- [ ] Confirm active error alert rules are configured.
- [ ] Connect live slack alerts channel webhook targets.
- [ ] Verify that real-time database snapshots sync with the dashboard.
