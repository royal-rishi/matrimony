# Broadcast Campaign Management Guide

This guide explains how to design, segment, and execute bulk marketing campaigns in the **RishtaJodo Matrimony** admin portal.

---

## 1. Segmented Audiences

Campaigns can be targeted to specific subsets of the user directory:

*   **Genders**: Male, Female, or All.
*   **Verification Status**: Verified profiles only, unverified profiles, or all profiles.
*   **Estimated Reach**: The wizard executes a count query on profiles matching filters before launching to calculate target metrics.

---

## 2. Dispatching Broadcasts

1.  Open the **Broadcast Wizard** tab.
2.  Name the campaign and select a template.
3.  Choose target filters. The wizard updates the **Estimated Reach** count.
4.  Click **Create Campaign Draft**.
5.  Find the draft in the campaign table and click **Broadcast Now** to queue the bulk dispatches.

---

## 3. Campaign Performance Tracking

The campaign list displays delivery metrics after dispatch:
*   **Status**: Draft $\rightarrow$ Processing $\rightarrow$ Completed.
*   **Target Reach**: Total matching audience size.
*   **Success metrics**: Displays counts of sent vs failed messages.
