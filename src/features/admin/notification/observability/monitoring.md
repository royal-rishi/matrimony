# Real-Time Queue & Events Monitoring

Live queues and logging streams are updated automatically without manual refresh loops via Supabase Realtime subscriptions.

## Features
- **Queue Pending & processing count**: Visualizes channels backlog size.
- **Stuck queue items detection**: Alerts operators to jobs processing for more than 30 minutes.
- **Live scrolling log feed**: Slides new inserts into view instantly.
- **DLQ manual acknowledged resolution**: Operators can attach resolution notes and resolve failed notifications.
